Politique de Confidentialité

Dernière mise à jour : 21 avril 2026 · Version 1.0

En résumé : WASI Intelligence ne collecte pas vos données personnelles sur ses serveurs. Votre historique de chat et votre token d'accès sont stockés localement dans votre navigateur (localStorage). Vous pouvez les supprimer à tout moment.

1. Responsable du traitement

WASI — Whole African Strategic Intelligence

Contact : support@wasi.africa

2. Données collectées et finalités

Donnée Stockage Finalité Durée
Token d'accès WASI (wasi_access_token) localStorage navigateur Authentification au service IA Jusqu'à déconnexion / effacement
Historique de conversation (wasi_chat_history_v2) localStorage navigateur Continuité du contexte entre sessions 30 derniers messages (glissant)

Messages envoyés à l'IA Transmis au proxy backend (Render.com, États-Unis) Génération de réponses via l'API Claude (Anthropic) Non conservés côté serveur

WASI Intelligence ne collecte pas : nom, email, adresse IP persistée, données de paiement, cookies tiers, données de navigation.

3. Base légale du traitement (RGPD art. 6)

  • Intérêt légitime (art. 6.1.f) — stockage local pour la continuité de service
  • Consentement (art. 6.1.a) — pour tout traitement au-delà du fonctionnement strict du service
  • Exécution d'un contrat (art. 6.1.b) — traitement des messages pour fournir le service d'intelligence souscrit

4. Transferts de données

Vos messages sont transmis à notre proxy sécurisé hébergé sur Render.com (San Francisco, États-Unis), qui les transmet à l'API Anthropic Claude pour générer une réponse. Ces transferts sont couverts par les Clauses Contractuelles Types (CCT) de la Commission européenne.

La clé API Anthropic n'est jamais transmise à votre navigateur — elle réside exclusivement dans les variables d'environnement du serveur proxy.

5. Vos droits

Conformément au RGPD (Règlement UE 2016/679) et au cadre de protection des données de la CEDEAO, vous disposez des droits suivants :

  • Accès — vos données sont dans votre localStorage ; vous pouvez les consulter via les outils développeur de votre navigateur
  • Effacement — cliquez sur "Effacer l'historique" dans l'interface, ou effacez le localStorage depuis votre navigateur
  • Portabilité — exportable via les outils développeur
  • Opposition — refusez le consentement RGPD dans la bannière ; les données locales seront supprimées
  • Réclamation — vous pouvez saisir l'autorité de protection des données de votre pays membre de l'UE ou la CNIL (France)

Pour exercer vos droits : support@wasi.africa

6. Sécurité

  • Toutes les communications utilisent HTTPS (TLS 1.3)
  • La clé API est stockée exclusivement côté serveur (variables d'environnement Render)
  • Le proxy backend valide un token d'accès WASI avant chaque requête
  • Rate limiting : 30 requêtes par minute par token
  • CORS restreint au domaine GitHub Pages officiel

7. Cookies

WASI Intelligence n'utilise aucun cookie. Le stockage des données s'effectue exclusivement via l'API localStorage du navigateur, qui reste sur votre appareil et n'est pas transmis automatiquement au serveur.

8. Mineurs

WASI Intelligence est une plateforme d'intelligence économique professionnelle destinée aux personnes de 18 ans et plus. Nous ne collectons pas sciemment de données relatives à des mineurs.

9. Modifications

Toute modification substantielle de cette politique sera notifiée via une bannière sur la plateforme. La date de dernière mise à jour figure en haut de cette page.
